## Run Locally
Clone the project

```bash
  git clone https://github.com/d3m0n2/aimlabs-stream-overlay
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
